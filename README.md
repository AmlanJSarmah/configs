# configs
My window manager and neovim configuration



## Requirements 

1. zsh shell with oh my zsh with powerlevel10k theme
2. qtile window manager (psutils and iwlib)
3. neovim with packer
4. picom
5. rofi
6. a valid nerd font
7. dmenu
